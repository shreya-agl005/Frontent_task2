#  Frookoon – Static Frontend Page
This project is a **static frontend page** created as part of an internship task for **Frookoon**
##  Tech Stack

- React.js (Vite)
- CSS
  
## Components
  Header.jsx : Navigation bar with logo and menu
  Hero.jsx : Tagline section
  Footer.jsx : Footer section

## Pages
  Home.jsx → Home screen 
  ProductListing.jsx → List of products 
  Cart.jsx → Cart screen 
  OrderConfirmation.jsx → Order confirmation screen
